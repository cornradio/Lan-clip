# Release Notes

## v1.7.1 - 2026-07-08
- Fix: macOS AppKit Dock 图标隐藏仅在 macOS 环境运行，不干扰其他平台
- Fix: 卡片视图打开时直接加载为卡片模式，不再先从列表模式闪烁切换
- Fix: 图片/文件粘贴在卡片模式下正确渲染，不再显示为 HTML 标签（Jinja2 safe_or_escape 过滤器修复）
- Fix: 卡片模式下编辑功能弹出悬浮窗口，不再导致内容丢失
- UI: 卡片样式优化 — 4:3 比例、毛玻璃新建卡片、2px 半透明边框、去除背景和渐变遮罩
- UI: 新建卡片从内联展开改为居中悬浮弹窗
- UI: 标题改为 LAN CLIP 全大写，设置中显示版本号
- UI: 工具栏按钮默认隐藏，设置中可逐个开关（回到顶部/切换视图/选中模式/图册预览）
- UI: 清空历史记录从 toggle 改为红色按钮，放在设置最底部
- 新增: releasenote.md 独立更新日志

## 2026-07-07
1. 增加一个按钮 点击就是"粘贴并发送"功能，比如空文本发送时候直接发送剪切板文本
2. 文本渲染要增加 md 支持
3. 除了系统自带的之外，不允许渲染 HTML
4. 增加了 gridview，一种类似 Google Keep Notes 的卡片视图

## 2026-02-28
- 增加一个权限管理功能，置顶、编辑、删除帖子需要密码（可在 pwd.txt 中修改密码）

## 2026-02-28
- 图册预览给每个图增加下载按钮、删除按钮
- "设置 → 简洁模式"开关之间切换，对比体验：普通模式保留动效和大图展示，简洁模式更平、滚动更快、操作按钮更突出
- 上传进度条，UPLOAD_PROGRESS_MIN_SIZE 以上则显示进度提示
- pwd.txt 增加密码设置
- 删除动画
- 双击卡片就开始进入高亮卡片模式，上下左右控制选择，del/backspace 可以删除（删除后自动选择下一个卡），d 下载，e 编辑，c 复制（仅在高亮卡片时候有效果）
